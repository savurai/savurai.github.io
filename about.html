About This Project

This AI-powered detection system was built using a combination of modern tools:

  • Roboflow — dataset management + model training
  • Flutter — cross-platform UI framework
  • FastAPI — backend server
  • The website version uses HTML, CSS, and JavaScript for live inference

The YOLO-based model detects components in real time using your webcam. Bounding boxes and confidence scores are drawn directly on the canvas with no delays.
