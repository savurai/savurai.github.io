About This App

This application uses computer vision technology to detect objects in real time using your device’s camera.

How it was built

  • ✔ Trained using Roboflow
  • ✔ Inference powered by Roboflow Hosted API
  • ✔ Frontend built with HTML, CSS, and JavaScript
  • ✔ Canvas used for drawing bounding boxes

Designed and developed for learning, experimentation, and AI computer vision demos.
