How It Was Built

VisionAI is built using a combination of modern AI and web technologies:

  • Roboflow – dataset creation, annotation, augmentation, and model hosting.
  • YOLO11 – state-of-the-art object detection architecture.
  • JavaScript + WebRTC – for live webcam video streaming.
  • HTML/CSS – to create this clean, responsive UI.

The detector supports 16 electronic component classes including transistors, capacitors, LEDs, heatsinks, diodes, and microprocessors.
