About VisionAI

VisionAI is a browser-based object detection app built using:

  • Roboflow – for training the detection model
  • Roboflow Inference SDK – for running the model
  • WebRTC getUserMedia – for accessing the webcam
  • HTML / CSS / JavaScript – for building the UI

The custom model used in this project is trained specifically for your categories and deployed using the Roboflow Serverless API.
